🧪 Widget Embed Test Page

Simulating an external website using the donation widget

📱 Modal Mode (Recommended)

Click the button below to open the donation widget in a modal overlay.

<!-- Modal Mode Implementation --> <button id="donate-modal-button">Donate Now</button> <script src="/widget-loader.js"></script> <script> DonationWidget.init({ siteId: 'demo-site', trigger: '#donate-modal-button', mode: 'modal', onSuccess: function(data) { console.log('Donation successful!', data); } }); </script>

🖼️ Inline Mode

The widget is embedded directly into the page content.

<!-- Inline Mode Implementation --> <div id="inline-widget-container"></div> <script src="/widget-loader.js"></script> <script> DonationWidget.init({ siteId: 'demo-site', targetElement: '#inline-widget-container', mode: 'inline' }); </script>

✨ Features

  • iframe-based isolation (styles won't conflict with host page)
  • postMessage communication between host and widget
  • Modal and inline display modes
  • Customizable theme and branding
  • Event callbacks (onSuccess, onError, onClose)
  • Responsive design (works on mobile and desktop)
  • Keyboard shortcuts (ESC to close modal)

📊 Widget Events

[Init] Console ready. Widget events will appear here.

⚙️ Configuration Options

Option Type Description
siteId string Required. Your unique site identifier
mode 'modal' | 'inline' Display mode. Default: 'modal'
trigger string CSS selector for trigger element (modal mode)
targetElement string CSS selector for container (inline mode)
theme 'light' | 'dark' Color theme. Default: 'light'
onSuccess function Callback when donation succeeds
